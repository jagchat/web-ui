Create Vue App project (with webpack-simple template)

	• Create folder "vapps" (this is not app, this is to dowonload vue client locally in the folder, if not in global)
	• >cd vapps
	• >npm init -y
	• >npm install --save-dev @vue/cli@3.0.4
	• >npm install --save-dev @vue/cli-init@3.0.4
	• >.\node_modules\.bin\vue init webpack-simple app01
	• Accept defaults (keep on pressing enter till you get prompt)
	• >cd app01
	• >npm install
	• >npm run dev
